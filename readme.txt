WooCommerce Smart Payment Recovery + WhatsApp Reminder. Recover failed payments automatically.

== Description ==

PayRevive is a smart payment recovery solution for WooCommerce that helps you recover failed transactions through automated retries and multi-channel notifications (Email and WhatsApp).

=== Key Features ===
* **Smart Retry Logic**: Automated payment retries using Action Scheduler with exponential backoff support.
* **Email Reminders**: Customizable email templates to nudge customers to complete their payment.
* **WhatsApp Integration**: Send recovery reminders directly to customers' WhatsApp (simulated/framework provided).
* **Modern Dashboard**: Professional analytics dashboard to track recovery rates and revenue saved.
* **HPOS Compatible**: Fully compatible with WooCommerce High-Performance Order Storage.

== Installation ==

1. Upload the `payrevive` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'PayRevive' in the admin sidebar to configure your settings.
